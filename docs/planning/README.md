# docs/planning/ — agent planning & orchestration scratch

This directory holds **agent planning and orchestration documents** — task breakdowns,
implementation plans, scratch notes, multi-agent coordination files, etc.

**Everything here is gitignored except this README** (see `.gitignore`). These files are
local working artifacts, not part of the repository's source of truth:

- Durable decisions belong in `docs/DISCOVERY.md`.
- Standards and conventions belong in `docs/ENGINEERING.md`.
- The product/template/API contract belongs in `docs/SPEC.md`.

Treat anything in this folder as disposable.
